This page documents all `bc_*` configuration variables related to BestClient's social systems and utility tooling — inline media previews in chat, the Client Indicator presence system, server browser enhancements, streamer mode privacy options, the admin panel, and several smaller convenience features. All variables are saved to `settings_ddnet.cfg` unless noted otherwise.

---

## Chat Media

BestClient can fetch and render image and GIF previews directly inside the chat panel when a recognized URL is posted.

<ParamField query="bc_chat_media_preview" type="int" default="1">
  Master toggle for rendering media previews from URLs posted in chat. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_chat_media_photos" type="int" default="1">
  Render static image (photo) previews from chat links. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_chat_media_gifs" type="int" default="1">
  Render animated GIF previews from chat links. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_chat_media_content_filter" type="int" default="1">
  Only allow media previews from domains listed in `bc_chat_media_allowed_domains`. `0` = allow all domains, `1` = allowlist only.
</ParamField>

<ParamField query="bc_chat_media_allowed_domains" type="string" default="tenor.com; imgur.com; giphy.com">
  Semicolon-separated list of domains from which media previews are permitted when the content filter is enabled. Example: `tenor.com; imgur.com; giphy.com`.
</ParamField>

<ParamField query="bc_chat_media_preview_max_width" type="int" default="220">
  Maximum width of a rendered chat media preview in pixels. Range: `120`–`400`.
</ParamField>

<ParamField query="bc_chat_media_viewer" type="int" default="1">
  Enable a fullscreen media viewer that opens when you click a chat preview. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_chat_media_viewer_max_zoom" type="int" default="800">
  Maximum zoom level available in the fullscreen media viewer, in percent. Range: `100`–`2000`.
</ParamField>

## Chat Behavior

<ParamField query="bc_chat_save_draft" type="int" default="0">
  Preserve unsent chat input when you close the chat box, so it is still there when you reopen it. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_silent_typing" type="int" default="0">
  Hide the `PLAYERFLAG_CHATTING` flag from other players while you are typing, so the chat bubble does not appear above your tee. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_chat_alt_command_layout" type="int" default="0">
  Recognize slash commands typed using a Russian (QWERTY-equivalent) keyboard layout. Useful for players whose OS default layout is Russian. `0` = off, `1` = on.
</ParamField>

## Client Indicator

The Client Indicator system broadcasts your presence to a lightweight UDP server and annotates name plates and the scoreboard to show which players are running BestClient.

<ParamField query="bc_client_indicator" type="int" default="1">
  Client Indicator is always enabled; this variable documents that state. Fixed at `1`.
</ParamField>

<ParamField query="bc_client_indicator_versions" type="int" default="0">
  Show a Versions tab in the BestClient server browser panel listing which version each player is running. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_client_indicator_in_name_plate" type="int" default="1">
  Show the BestClient indicator icon in player name plates. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_client_indicator_in_name_plate_above_self" type="int" default="0">
  Show the client indicator above your own tee's name plate as well. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_client_indicator_in_name_plate_size" type="int" default="30">
  Relative size of the indicator icon in name plates. Range: `-50`–`100`.
</ParamField>

<ParamField query="bc_client_indicator_in_name_plate_dynamic" type="int" default="1">
  Allow the indicator icon to reposition dynamically in name plates based on other nameplate content. `0` = fixed, `1` = dynamic.
</ParamField>

<ParamField query="bc_client_indicator_in_scoreboard" type="int" default="1">
  Show the BestClient indicator icon in the scoreboard next to each player's name. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_client_indicator_in_scoreboard_size" type="int" default="100">
  Relative size of the indicator icon in the scoreboard. Range: `-50`–`100`.
</ParamField>

<ParamField query="bc_nameplate_client_indicator_offset_x" type="int" default="0">
  Horizontal pixel offset for the client indicator icon within nameplates. Range: `-400`–`400`.
</ParamField>

<ParamField query="bc_nameplate_client_indicator_offset_y" type="int" default="0">
  Vertical pixel offset for the client indicator icon within nameplates. Range: `-400`–`400`.
</ParamField>

<ParamField query="bc_client_indicator_server_address" type="string" default="150.241.70.188:8778">
  UDP address of the Client Indicator presence server. Change only if you are running a self-hosted instance.
</ParamField>

<ParamField query="bc_client_indicator_browser_url" type="string" default="https://150.241.70.188:8779/users.json">
  HTTPS URL that the server browser queries to fetch the list of online BestClient users.
</ParamField>

<ParamField query="bc_client_indicator_token_url" type="string" default="https://150.241.70.188:8779/token.json">
  HTTPS URL used to bootstrap the shared authentication token on first run.
</ParamField>

<ParamField query="bc_client_indicator_shared_token" type="string" default="">
  Shared token appended to signed UDP presence packets. Auto-populated from `bc_client_indicator_token_url` on first launch.
</ParamField>

<ParamField query="bc_client_indicator_secret_key" type="string" default="">
  Developer secret key for advanced Client Indicator API access. Leave empty unless instructed otherwise.
</ParamField>

## Streamer Mode

Streamer mode hides sensitive information from your screen using a bitmask stored in `bc_streamer_flags`. Each bit independently controls one category of hidden information.

<ParamField query="bc_streamer_flags" type="int" default="64">
  Bitmask of active streamer mode options. Combine values by adding them together. Default `64` hides the login panel only. Range: `0`–`127`.

  | Value | Flag | Effect |
  |-------|------|--------|
  | `1` | `STREAMER_HIDE_SERVER_IP` | Hide the server IP address |
  | `2` | `STREAMER_HIDE_CHAT` | Hide all chat messages |
  | `4` | `STREAMER_HIDE_FRIEND_WHISPER` | Hide friend whisper messages |
  | `8` | `STREAMER_HIDE_OWN_NAME` | Hide your own player name |
  | `16` | `STREAMER_HIDE_OTHER_NAMES` | Hide all other players' names |
  | `32` | `STREAMER_HIDE_TAB_NAMES` | Hide names in the scoreboard tab |
  | `64` | `STREAMER_HIDE_LOGIN` | Hide the login/password panel |
</ParamField>

## Components Mask

Advanced: these two integers form a 62-bit bitmask that disables specific BestClient components. Each bit corresponds to one entry in the `EBestClientComponent` enum. Use the BestClient settings UI rather than editing these directly.

<ParamField query="bc_disabled_components_mask_lo" type="int" default="0">
  Low 31 bits of the disabled-components bitmask (components 0–30). Range: `0`–`2147483647`.
</ParamField>

<ParamField query="bc_disabled_components_mask_hi" type="int" default="0">
  High 31 bits of the disabled-components bitmask (components 31–61). Range: `0`–`2147483647`.
</ParamField>
