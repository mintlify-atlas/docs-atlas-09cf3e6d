BestClient ships with a rich set of visual enhancements — from subtle UI animations and eye-comfort overlays to full particle systems and tee deformation effects. Every option listed here can be changed live from the in-game console or configured through the BestClient settings UI. All variables are persisted to `settings_ddnet.cfg` in your `$USERDIR`.

---

## Aspect Ratio

Control the custom aspect ratio feature that lets you render at a non-native ratio such as 4:3 or 5:4.

<ParamField query="bc_custom_aspect_ratio_mode" type="int" default="-1">
  Aspect ratio mode. `-1` = legacy auto, `0` = off, `1` = preset, `2` = custom.
</ParamField>

<ParamField query="bc_custom_aspect_ratio_apply_mode" type="int" default="1">
  Where the custom aspect ratio is applied. `0` = game viewport only, `1` = full screen, `2` = game viewport without HUD. Range: `0`–`2`.
</ParamField>

<ParamField query="bc_custom_aspect_ratio" type="int" default="0">
  Aspect ratio value multiplied by 100. `0` = off. Common presets: `125` = 5:4, `133` = 4:3, `150` = 3:2. Accepts any value from `0` to `1000`.
</ParamField>

<ParamField query="bc_custom_aspect_ratio_num" type="int" default="0">
  Custom aspect ratio numerator (width component). `0` = unset. Range: `0`–`100000`.
</ParamField>

<ParamField query="bc_custom_aspect_ratio_den" type="int" default="0">
  Custom aspect ratio denominator (height component). `0` = unset. Range: `0`–`100000`.
</ParamField>

<ParamField query="bc_3d_particles_size_max" type="int" default="8">

<ParamField query="bc_3d_particles_speed" type="int" default="18">
  Base movement speed of 3D particles. Range: `1`–`500`.
</ParamField>

<ParamField query="bc_3d_particles_depth" type="int" default="300">

</ParamField>

<ParamField query="bc_3d_particles_alpha" type="int" default="35">

<ParamField query="bc_3d_particles_push_radius" type="int" default="120">
  Radius around the local player that pushes nearby particles away. Range: `0`–`1000`.

<ParamField query="bc_3d_particles_push_strength" type="int" default="120">

## Music Player

</ParamField>

<ParamField query="bc_music_player_visualizer_smoothing" type="int" default="50">

</ParamField>

<ParamField query="bc_music_player_visualizer_rounding" type="int" default="0">

<ParamField query="bc_music_player_visualizer_columns" type="int" default="5">

</ParamField>

<ParamField query="bc_music_player_visualizer_column_width" type="int" default="100">

<ParamField query="bc_music_player_visualizer_gap" type="int" default="100">

## Optimizer

The Optimizer reduces GPU/CPU load with configurable culling and process priority adjustments.

<Accordion title="Optimizer variables (10 variables)">

<ParamField query="bc_optimizer" type="int" default="0">
  Enable optimizer features. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_optimizer_disable_particles" type="int" default="0">
  Disable rendering and updating all particles. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_optimizer_fps_fog" type="int" default="0">
  Cull non-map rendering outside a distance limit around the camera. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_optimizer_ddnet_priority_high" type="int" default="0">
  Set the DDNet process priority to High while the optimizer is enabled. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_optimizer_discord_priority_below_normal" type="int" default="0">
  Set the Discord process priority to Below Normal while the optimizer is enabled. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_optimizer_fps_fog_mode" type="int" default="0">
  FPS fog culling mode. `0` = manual radius in tiles, `1` = by zoom percent. Range: `0`–`1`.
</ParamField>

<ParamField query="bc_optimizer_fps_fog_radius_tiles" type="int" default="40">
  Manual FPS fog radius in tiles (1 tile = 32 units). Used when `bc_optimizer_fps_fog_mode` is `0`. Range: `5`–`300`.
</ParamField>

<ParamField query="bc_optimizer_fps_fog_zoom_percent" type="int" default="70">
  Visible area percentage in zoom mode. Used when `bc_optimizer_fps_fog_mode` is `1`. Range: `10`–`120`.
</ParamField>

<ParamField query="bc_optimizer_fps_fog_render_rect" type="int" default="0">
  Render an outline rectangle showing the active FPS fog area. `0` = off, `1` = on.
</ParamField>

<ParamField query="bc_optimizer_fps_fog_cull_map_tiles" type="int" default="0">
  Also cull map tile rendering outside the FPS fog area. `0` = off, `1` = on.
</ParamField>

</Accordion>
