The Fun tab is a curated section of the BestClient main menu that bundles 14 fully playable mini-games directly inside the client. Whether you are waiting for a map download, taking a break between runs, or just want something to do in the lobby, every game in the Fun tab is available instantly — no extra installs, no config variables, and no network connection required. All games run inside the BestClient UI layer and can be opened and closed with a single click at any time.

## Accessing the Fun Tab

Click **Fun** in the BestClient main menu sidebar. The tab is present from the first launch and requires no setup. All 14 games are unlocked and available immediately.

## No Configuration Required

The Fun tab has no associated `bc_*` config variables. Every game is always enabled and cannot be disabled through the Components Editor or any other setting. There is nothing to configure — open the tab and play.

## Tips

- Fun tab games run entirely in the client UI, so they work in the main menu, in spectator, or while connected to a server in the background.
- Your score and game state are not persisted between sessions — each game starts fresh when you open it.
- The casino balance (`bc_casino_balance`) used by the in-client Casino game is separate from the Fun tab and has its own config variable.
